# MIOKAのお手伝い帳 🌟

MIOKAちゃんのお手伝いお小遣いを記録するアプリです。

## お手伝いリスト

| お手伝い | お小遣い |
|---------|---------|
| お風呂掃除 | 10円 |
| 洗濯物畳む | 10円 |
| ここちゃんのお世話 | 10円 |
| ご飯作り | 10円 |
| その他 | 10円 |

## 使い方

1. `index.html` をダブルクリックしてブラウザで開く
2. **きょう** タブで、やったお手伝いをタップしてチェック
3. **こんげつ** タブで今月の合計を確認
4. 月末に「今月を清算する」ボタンを押して清算

データはブラウザに自動保存されます（同じパソコン・同じブラウザで使ってください）。
